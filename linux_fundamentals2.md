## common directories

``` 
root/
├── etc/ (OS Configuration) - store files that are used by OS
├── var/ (Variable Data/Logs) - stores data that is frequently accessed or written by services or applications
├── root/ (Admin Home) - home for the **root** system user
└── tmp/ (Temporary Files) - store data that is only needed to be accessed once or twice
```
